# listaVuelos
Usando Java stream.

Filtramos una lista de vuelos entre dos fechas y mostramos por terminal el resultado.

El filtro permite:

Buscar vuelos entre dos fechas (fechaInicio y fechaFin).

Buscar vuelos desde una fecha en adelante.

Buscar vuelos hasta una fecha determinada.

Devolver todos los vuelos si no se pasa ninguna fecha.
